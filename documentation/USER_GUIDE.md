## üéÆ Getting Started

### First Time Setup

1. **Access the Game**
   - Open your browser to `http://localhost:5173`
   - You'll see the Nexus Protocol landing page

2. **Create or Join a Team**
   - Click "ENTER NEXUS" or navigate to `/login`
   - Enter your team name and access code
   - Demo credentials: Team `Ghost`, Code `1234`

3. **Select Your Agent**
   - Choose from three agent roles
   - Each has unique abilities and playstyles

4. **Start Your First Mission**
   - Review available missions
   - Select one to begin
   - Complete objectives to earn points

## üìä Scoring System

### How Scoring Works

**Base Score:**
- Objective completion points
- Time bonus (faster = more points)
- Tool efficiency bonus

**Penalties:**
- Trace level (higher = more penalty)
- Alarms triggered
- Failed objectives

**Bonuses:**
- Perfect stealth (no trace)
- Speed run (under time)
- All objectives complete

### Rank System

| Rank | Score Range | Description |
|------|-------------|-------------|
| **S-RANK** | 9000+ | Perfect execution |
| **A-RANK** | 7500-8999 | Excellent performance |
| **B-RANK** | 6000-7499 | Good performance |
| **C-RANK** | 4500-5999 | Average performance |
| **D-RANK** | 3000-4499 | Below average |
| **F-RANK** | 0-2999 | Failed mission |

### Tips for High Scores

1. **Complete All Objectives**
   - Required objectives are mandatory
   - Optional objectives boost score

2. **Minimize Trace**
   - Use low-risk tools when possible
   - Clean traces regularly
   - Avoid triggering alarms

3. **Optimize Time**
   - Plan route before starting
   - Don't waste time on unnecessary actions
   - Use tools efficiently

4. **Tool Efficiency**
   - Use fewer tools for same result
   - Choose right tool for each task
   - Avoid redundant tool usage
